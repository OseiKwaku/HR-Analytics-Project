# HR-Analytics-Project

### Project Overview
This project presents an interactive Workforce Overview Dashboard designed to analyze employee data, focusing on headcount, attrition, compensation, and performance metrics. The goal is to provide actionable HR insights for better workforce planning and decision-making.

### Objectives
Monitor workforce size and structure
Analyze employee attrition trends
Evaluate compensation distribution
Assess employee performance and engagement
Identify departmental workforce patterns

### Dataset
The dataset includes HR-related attributes such as:
Employee demographics (age groups, gender)
Departmental distribution
Salary and compensation data
Performance ratings
Attrition status and tenure

### Tools & Technologies
Power BI – Data visualization & dashboard creation
DAX (Data Analysis Expressions) – Calculated measures and KPIs
Excel / CSV – Data storage and preprocessing

### Key Metrics
Total Employees: 311
Active Employees: 207
Attrition Rate: 50.2%
Average Tenure: 9 years
Average Salary: $69.02K
Total Salary: $21.47M
Average Engagement Score: 4.11

### Key Insights
High attrition rate (50.2%) suggests potential retention challenges.
Production department has the highest workforce and also the highest attrition.
Majority of employees fall within Gen X and Boomers, indicating an aging workforce.
Performance distribution shows a concentration in “Needs Improvement” and “PIP”, highlighting productivity concerns.
Salary distribution appears balanced but may require alignment with performance outcomes.

### Dashboard Features
Workforce summary KPIs
Employee distribution by department
Attrition analysis by department
Age group segmentation
Performance analysis by gender
Salary and engagement insights

### Business Value
This dashboard enables HR teams and decision-makers to:
Identify retention risks early
Optimize workforce allocation
Improve performance management strategies
Support data-driven HR policies
